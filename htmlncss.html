1. Apa itu HTML?

HTML (singkatan dari HyperText Markup Language) adalah bahasa markah standar yang digunakan untuk membuat dan menyusun halaman web. HTML digunakan untuk menentukan struktur konten di dalam sebuah halaman

2. Apa itu CSS?

CSS (singkatan dari Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan gaya dari elemen-elemen HTML di sebuah halaman web. Kalau HTML adalah struktur, maka CSS adalah penampilannya â€” seperti warna, ukuran teks, jarak antar elemen, posisi layout, dan animasi.

3. Struktur Dasar HTML

HTML (HyperText Markup Language) memiliki struktur dasar yang digunakan untuk membangun halaman web

4. Tag Heading dan Paragraf

Gunakan tag <h1> sampai <h6> untuk judul, dan <p> untuk paragraf.

5. Link dan Gambar

Gunakan tag <a> untuk link dan <img> untuk gambar.

6. List dan Tabel

Gunakan <ul> atau <ol> untuk daftar, dan <table> untuk tabel.

7. Formulir HTML

Gunakan tag <form> untuk membuat formulir input.

8. Dasar CSS: Warna dan Font

CSS memungkinkan pengaturan warna, ukuran teks, dan font.

9. Box Model di CSS

Box model mencakup margin, border, padding, dan content.

10. Layout dengan Flexbox

Flexbox membantu membuat layout responsif dengan mudah.
